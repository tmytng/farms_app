# README

## 概要
### FARMSとは？
FARMS（ファームス）は1人〜15人前後の小規模チームを想定した営業支援ツールです。
チームメンバーそれぞれが日々の顧客とのやりとりを記録していくことで、
その情報をチーム全体で共有することができます。<br>
みんなで連携しながら、どんどん顧客発掘を行い、案件獲得に繋げていきましょう！
